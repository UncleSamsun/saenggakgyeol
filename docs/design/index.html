Brand / Design System v1

조용하게, 생각의 시간 맥락을 되찾는 Mac 작업 표면.

생각결의 디자인은 노트앱의 귀여움이나 AI 제품의 과한 장식보다, 세션 중 사용자가 남긴 생각과 transcript가 만나는 순간을 정확하게 보여주는 데 집중합니다.

Direction

기준은 Quiet Native Workspace입니다. Mac 메뉴바에 있어도 낯설지 않고, Obsidian으로 넘어가도 과장되지 않는 차분한 생산성 도구여야 합니다.

Memorable thing

조용한 맥락 복원. 처음 본 사람이 “내 생각이 나온 시간을 되찾아주는 앱”으로 기억해야 합니다.

Safe choices

Native density, readable editor surfaces, restrained color, clear status. Mac 앱의 기본 신뢰감을 빌립니다.

Creative risk

브랜드는 한국어 이름과 결/시간선 심볼로 갑니다. AI sparkle, brain, microphone 문법에서 벗어납니다.

Logo

심볼은 두 개의 시간선입니다. 하나는 사용자가 직접 쓴 note chunk, 하나는 녹음에서 나온 transcript chunk입니다. 중앙의 노드는 둘이 맞물리며 맥락이 복원되는 순간입니다.

결/시간선 심볼

마이크, 뇌, 말풍선, sparkle은 쓰지 않습니다. 생각결은 음성노트나 챗봇보다, 사용자가 직접 남긴 생각의 시간 맥락을 다루는 제품입니다.

note line transcript line context node D14 approved

Small-size rule

16px에서는 중앙 노드와 두 선의 방향만 남아도 충분합니다. 작은 크기에서 텍스트나 복잡한 디테일을 억지로 유지하지 않습니다.

Color

색은 많지 않게, 의미 있게 씁니다. Primary는 active session, Context Blue는 transcript/evidence, Warm Signal은 decision/important에 둡니다.

Component Design rule Implementation note
Menu bar popover Compact, status-first, one primary action. 360-420px target width.
Floating session editor Small launch, real long-form editor. Hide side panels before shrinking writing surface.
Transcript evidence Context Blue, timestamp range, short quote. Always link back to note chunk.
AI synthesis status Never outrank the user's original notes. Idle, queued, processing, completed, failed.
